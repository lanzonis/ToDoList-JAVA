# ToDoList-JAVA
 Lista simples de prioridades realizada em JAVA

Criado em JAVA, SWING e AWT como trabalho para o quarto semestre da faculdade (UNIP), com a conexão JDBC para inserir os itens da lista em um banco de dados.

*código comentado.

Sobre o projeto:
Parte gráfica realizada no SWING utilizando os componentes gráficos da GUI para fazer o front do projeto. A lista contém 3 prioridades (ALTA, MÉDIA e BAIXA) o usuário pode, mover, adicionar e excluir cada item de uma prioridade.

Sobre a Conexão do Banco de dados:
Criado um BANCO em POSTGRES e utilizando DBEAVER como SGBD.
Utilizei JDBC para realizar a inserção ao banco de dados, essa parte não era necessária para a conclusão do projeto.
